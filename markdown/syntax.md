# markdown语法

## 标题

```markdown
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

效果如下:

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

## 字体

```markdown
**这是加粗的文字**

*这是倾斜的文字*

***这是斜体加粗的文字***

~~这是加删除线的文字~~
```

效果如下:

**这是加粗的文字**

*这是倾斜的文字*

***这是斜体加粗的文字***

~~这是加删除线的文字~~

## 代码

代码块使用 ```进行包裹

```markdown
    ```java
        public class Person{
            private int age;
            private String name;
        }
    ```
```

效果如下:

```java
public class Person{
    private int age;
    private String name;
}
```
